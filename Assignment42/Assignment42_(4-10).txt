

4. Why is KNN called a lazy learner?
Ans: KNN is simple Supervised Machine Learning Algorithm used for Both Classification and Regression.
KNN widely used for Classification.

KNN works by:
1)Storing all Data points.
2)calculate new point Distance with respect to this point using Euclidian Distance
3)Do Classification or Regression Based on the Data

In KNN,all the calculations happens while Predicting the ouput.While Traning no Calculations done
thats why KNN is called Lazy Learner.
